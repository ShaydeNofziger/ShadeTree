    Reading your tree

    🌿 Placement
    Each discipline grows its own branch. Leaves appear in the order you logged them — earliest jumps sit near the trunk, later ones march toward the tip. A seeded random scatter spreads leaves sideways so the branch fills out naturally. The same logbook always draws the same tree.
    🟢 Size
    Leaf radius scales with freefall distance (exit altitude minus deployment altitude). Longer freefalls produce bigger leaves.
    🎨 Color
    Each leaf is colored by the season it was jumped in — spring, summer, autumn, or winter.
    ✨ Milestones
    Gold-ringed leaves mark milestones: jump #100, #500, #1,000, first jump in a new discipline, and your last student jump.
    🌳 Trunk
    Trunk height grows with the number of seasons you've been jumping.
